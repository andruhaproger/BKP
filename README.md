## Основная идея исследования

В работе сравнивались несколько алгоритмов машинного обучения для задачи регрессии:
- Linear Regression
- Ridge Regression
- Lasso
- ElasticNet
- Random Forest
- Extra Trees
- HistGradientBoosting
- CatBoost

Для линейных моделей использовалось **TF-IDF-представление** свободных текстовых ответов учащихся.

Для ансамблевых моделей использовались **компактные ручные текстовые признаки**, такие как:
- длина текста;
- число слов;
- число уникальных слов;
- тематические индикаторы.

Отдельно был проведён **аудит признаков на утечку целевой переменной**, чтобы итоговые метрики не были искусственно завышены.

Также был выполнен **кластерный анализ учащихся** по контекстным признакам. Кластеризация использовалась прежде всего как инструмент интерпретации структуры выборки.

---

## Основной результат

Наилучший честный результат показала модель **Ridge Regression** на безопасных табличных признаках и TF-IDF-признаках текста.

Лучшие метрики:
- **MAE = 6.0728**
- **RMSE = 7.7057**
- **R² = 0.7598**

Полученные результаты показали, что свободные текстовые ответы учащихся содержат значимый прогностический сигнал и существенно улучшают качество модели.
